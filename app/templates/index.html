Diabetes Detector

About

This web application can help detect whether a patient has diabetes. Using machine learning algorithms, the diabetes detector can identify diabetes based on the patient's data here.

The detector will make a prediction based on blood pressure, cholesterol level, BMI, smoking habit, disease history, physical activity, alcohol consumption, mental/physical health, age, education, and income.

XGBoost(Extreme Gradient Boosting): a scalable, distributed gradient-boosted decision tree machine learning algorithm which provides parallel tree boosting for regression, classification, and ranking.

KNN(K-Nearest Neighbors): a non-parametric, supervised learning classifier, which uses proximity to make classifications or predictions about the grouping of an individual data point.

Conclusion

From the result table, Random Forest has the highest training and testing accuracy for detecting diabetes while Gaussian Naive Bayes has the lowest training and testing accuracy for detecting diabetes.

Also, Random Forest and XGBoost have higher training and testing AUC scores than KNN nad Gaussian Naive Bayes.

From the results above, we should use Random Forest or XGBoost methods for modeling the data and detecting diabetes.
